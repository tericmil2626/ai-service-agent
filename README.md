# AI Service Business Receptionist

AI-powered SMS receptionist for home service businesses (plumbing, HVAC, electrical).

## Features
- SMS AI receptionist
- Missed call recovery
- Appointment scheduling
- Email notifications
- SQLite database

## Environment Variables
```
SIGNALWIRE_PROJECT_ID=your_project_id
SIGNALWIRE_TOKEN=your_token
SIGNALWIRE_PHONE_NUMBER=+1234567890
SIGNALWIRE_SPACE=your-space.signalwire.com
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
BUSINESS_EMAIL=your-email@gmail.com
BUSINESS_NAME=Your Business Name
```

## Deploy to Railway
1. Push to GitHub
2. Connect Railway to repo
3. Add environment variables
4. Deploy

## Local Development
```bash
npm install
npm start
```
